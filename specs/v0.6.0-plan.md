## Goal

Advance Batoi Press toward the `1.0.0` compatibility gate by making static export output self-checking, documented, and safer to hand off for static hosting without changing the flat-file runtime or update package contract.

## Release Theme

Verified static export packages.

## Out of Scope for v0.6.0

- Full media file copying inside static exports.
- Theme-system redesign.
- Mandatory signed release packages.
- Breaking changes to content files, theme templates, installer behavior, public routes, or update package structure.
