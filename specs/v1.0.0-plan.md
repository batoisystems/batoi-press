## Out of Scope for v1.0.0

- Mandatory database storage.
- Mandatory Composer, Node.js, Git, or CLI runtime requirements.
- Mandatory signed release packages.
- Batoi Platform workspace account dependency.
- Per-page ownership workflows.
- Multi-stage editorial approvals.
- Automatic media copying inside static export packages.

## Recommended First Implementation Slice

Start with documentation and release metadata. Runtime behavior reached the intended stable baseline in v0.9.0; v1.0.0 should avoid unnecessary code churn and focus on declaring the stable contract with reproducible evidence.
