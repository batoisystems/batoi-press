Status: Pending

- [x] Filter author post lists to the current user's authored posts.

## Out of Scope for v0.8.0

- Per-page ownership rules.
- Multi-author or reviewer workflows.
- Draft approval state machines.
- Breaking changes to post storage, content files, theme templates, public routes, or update package structure.
