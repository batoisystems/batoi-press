# Batoi UIF Components

Batoi Press bundles a small UIF runtime for admin and installer screens.

Use these primitives in admin views:

```text
bp-uif-surface
bp-uif-toolbar
bp-uif-button
bp-uif-button-secondary
bp-uif-notice
bp-uif-danger
bp-uif-table
bp-uif-field
```

The runtime assets are exposed publicly from:

```text
/assets/uif/uif.css
/assets/uif/uif.js
```

See `examples.html` for markup examples.
