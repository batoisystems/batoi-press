# Batoi Press 1.2.1

Batoi Press 1.2.1 completes the integration and responsive verification for the theme and site-branding implementation introduced in 1.2.0.

## Fixes

- Keeps page and post previews on their selected layouts when an installation has no published page or post content.
- Provides representative fallback page and post fixtures for empty-site theme previews.
- Preserves custom theme body classes while rendering the preview toolbar.

## Verification

- Covers logo storage, replacement, and removal through the validated branding pipeline.
- Covers theme package installation, same-slug upgrade, backup creation, and activation.
- Covers home, page, post, blog, archive, and 404 preview targets, including empty-content fallback behavior.
- Verifies text, logo, logo-plus-text, missing-logo fallback, long navigation, alternate themes, and desktop/mobile rendering in the synchronized testsite.
- Passes the complete asset, media, ownership, role access, security, smoke, static export, theme, syntax, and update test suite.
