# Batoi Press 1.2.2

Batoi Press 1.2.2 aligns the Users filter interface with the shared Admin Console filter pattern.

## Fixes

- Uses structured, explicitly associated fields for Search, Role, and Status filters.
- Aligns labels, inputs, selects, and actions on the desktop filter grid.
- Normalizes filter controls to a consistent 38-pixel border-box height across admin screens.
- Preserves a full-width single-column field layout and side-by-side actions on narrow screens without horizontal overflow.
