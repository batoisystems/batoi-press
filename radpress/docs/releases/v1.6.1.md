# Batoi Press 1.6.1

Batoi Press 1.6.1 is a focused follow-up to 1.6.0 that closes static-export, theme-editor, and featured-image accessibility gaps found during release review.

## Fixes

- Render a configured non-default homepage with root-relative static-export asset depth and the correct root sitemap URL.
- Allow missing optional theme CSS and JavaScript files to open with safe starter content and be created on save.
- Add featured-image alternative text to post editing, persistence, blog cards, individual posts, and static exports.
- Extend static-export regression coverage for configured homepages, sidebar widgets, recent posts, and featured-image accessibility.

## Versions

- Batoi Press: 1.6.1
- Bundled Batoi Versatile theme: 2.1.1
