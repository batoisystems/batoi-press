# Batoi Press 1.2.3

Batoi Press 1.2.3 keeps the Static Export action available so administrators can submit an export request and receive precise server-side diagnostics.

## Fixes

- Removes the disabled state from the Generate Export button when readiness checks detect missing ZIP support or directory permissions.
- Preserves ZIP support and storage status indicators before submission.
- Reports the exact missing capability or permission through the existing export failure response after submission.
- Adds regression coverage confirming the action remains clickable when readiness checks report a problem.

## Verification

- Generated a static export from the synchronized testsite through the browser.
- Verified a 404.1 KB ZIP containing 37 checked entries and all eight expected entries.
- Passed the complete asset, media, ownership, role access, security, smoke, static export, theme, syntax, and update test suite in both the repository and testsite.
