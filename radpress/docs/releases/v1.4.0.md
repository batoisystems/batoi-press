# Batoi Press 1.4.0

Batoi Press 1.4.0 ships Batoi Versatile, a substantially expanded default theme for company, service, editorial, campaign, and ecommerce presentation sites.

## Theme

- Rebuilds the bundled default theme with responsive navigation, accessible focus behavior, richer metadata, flexible content sections, improved typography, and a structured footer.
- Improves standard pages, posts, blog listings, archives, and the 404 experience.
- Adds reusable hero, section, feature, card, statistics, product, order-summary, form, account, and contact components.
- Includes responsive CSS and lightweight navigation, product-option, and quantity interactions as validated theme assets.

## Page Templates

- Activates the existing page `template` metadata field without changing the flat-file format.
- Adds manifest-declared page templates with safe layout-name validation and standard-page fallback.
- Adds page-template selection to the admin publishing panel.
- Adds landing, shop, product, cart, checkout, customer-account, and contact layouts.
- Adds complete preview fixtures for all bundled page templates.
- Preserves selected templates in live rendering and static exports.

## Ecommerce Scope

The ecommerce layouts provide content-managed storefront presentation and purchase handoff experiences. Inventory, pricing rules, persistent carts, customer authentication, payments, tax, shipping, and order processing are not included in this release.

## Compatibility

Existing pages continue using the standard page layout. Existing themes without page-template declarations normalize to that same layout. Missing, undeclared, or invalid template selections fall back safely without exposing arbitrary files.

## Verification

PHP syntax, theme validation, page-template persistence and fallback, static export, update, security, editor, smoke, desktop browser, mobile browser, responsive navigation, and image-loading checks passed.
