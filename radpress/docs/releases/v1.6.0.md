# Batoi Press 1.6.0

Batoi Press 1.6.0 completes the testing-team backlog for richer post presentation, operational theme controls, menu ordering, and reusable post sidebars.

## Highlights

- Add featured images to posts and render them in blog listings and individual articles.
- Create categories while editing posts and choose full-width, left-sidebar, or right-sidebar layouts.
- Add an editor-accessible widget manager with ordered, sanitized sidebar widgets and five recent posts by default.
- Reorder menu items directly in the menu editor.
- Improve theme action-button contrast and expose the bundled theme CSS and JavaScript in the constrained editor.
- Honor the configured homepage during static export.
- Update the bundled Batoi Versatile theme to version 2.1.0.

## Compatibility and Safety

- Homepage selection remains limited to published pages; posts and category archives remain content routes.
- Theme uploads continue to require a valid Batoi Press theme package. Legacy `templates/*.json` archives require an explicit migration rather than lossy automatic conversion.

## Verification

- PHP syntax checks pass for all changed PHP files.
- The complete repository test suite passes, including post-feature, editor-experience, static-export, security, theme, and update-runner checks.
